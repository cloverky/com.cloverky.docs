# 프로젝트명: 프리지 (Fridge)
> **AI 멀티 에이전트 기반 냉장고 재고 관리 및 개인화 레시피 서비스**

## 1. 에이전트 및 파일 구성 (Clean Version)
프로젝트 폴더 내에서 에이전트의 역할만 강조된 간결한 파일명 시스템입니다.

| 역할 | 파일명 | 핵심 기능 |
| :--- | :--- | :--- |
| **입고** | `manager.py` | 영수증/사진 기반 재고 및 유통기한 등록 |
| **실행** | `chef.py` | 유통기한 임박 재료 중심 레시피 제안 |
| **피드백** | `reviewer.py` | 사용자 취향 분석 및 Chef 로직 업데이트 |
| **보충** | `shopper.py` | 부족 재료 추출 및 장바구니 연동 |

---

## 2. 코드 주석용 가이드 (Docstring)
각 파이썬 파일 상단에 기술할 에이전트 정의서입니다.
```python
"""
[Agent: Manager]
- 역할: 입고 관리자
- 기능: 식재료 OCR 인식 및 유통기한 자동 DB 등록
"""

"""
[Agent: Chef]
- 역할: 실행 요리사
- 기능: 재고 맞춤형 레시피 생성 및 조리 가이드 제공
"""

"""
[Agent: Reviewer]
- 역할: 피드백 평가사
- 기능: 맛 취향 데이터화 및 개인화 추천 고도화
"""

"""
[Agent: Shopper]
- 역할: 구매 보충원
- 기능: 쇼핑 리스트 생성 및 이커머스 API 연동
"""
